unorderedOnEachParallel

inline fun <T> Flow<T>.unorderedOnEachParallel(concurrencyLevel: Int, crossinline f: suspend ConcurrencyInfo.(T) -> Unit): Flow<T>

Performs the provided f action concurrently on each item emitted by the flow. The action is applied with the specified concurrencyLevel. The order of items might not be preserved.

Return

A Flow of items with the action applied concurrently and possibly unordered.

Parameters

concurrencyLevel

The maximum number of concurrent invocations of the action f.

f

The action to apply to each item emitted by the flow.
